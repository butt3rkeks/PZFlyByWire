title=Helicopter Flight Engine Framework (B42)
description=Fixes helicopter flight for Build 42 and adds proper physics-based movement.
description=
description=[h1]What does this do?[/h1]
description=Helicopters in B42 use a broken movement system. This mod replaces it with real Bullet physics forces, giving you smooth, controllable flight with proper lift, descent, and momentum.
description=
description=Ships with a Fly-By-Wire (FBW) engine that handles everything: hover stability, tilt-based movement, yaw control, auto-leveling, and landing.
description=
description=[h1]Features[/h1]
description=[list]
description=[*] Stable hover, smooth ascent/descent
description=[*] Tilt to fly - arrow keys for pitch/roll, A/D for yaw, W/S for altitude
description=[*] Flight Assist Off mode (toggle in vehicle menu) for manual control
description=[*] Configurable via sandbox settings (speed, gravity, responsiveness)
description=[*] In-game tuning via /hef chat commands
description=[*] Wall collision detection prevents flying through buildings
description=[*] Engine damage and fuel consumption
description=[/list]
description=
description=[h1]Requirements[/h1]
description=[list]
description=[*] WarThunderVehicleLibrary
description=[*] UH-1B Helicopter (or other WTVL helicopters)
description=[/list]
description=
description=[h1]Replaces[/h1]
description=This replaces the old "UH-1B Helicopter B42 Fix" mod (3688683236). Unsubscribe from that mod when using this one.
description=
description=[h1]For modders[/h1]
